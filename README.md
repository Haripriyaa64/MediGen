# 🩺 MediGen AI

An AI-powered healthcare assistant that helps users analyze prescriptions, check symptoms, scan medicines, and interact with an intelligent medical chatbot.

## ✨ Features

- 🤖 AI Medical Chat
- 📄 Prescription Analysis (OCR + AI)
- 🩺 Symptom Checker
- 💊 Medicine Scanner
- 📑 PDF Report Generation

## 🛠 Tech Stack

**Frontend:** React, TypeScript, Vite, Tailwind CSS

**Backend:** FastAPI, Python, EasyOCR, Hugging Face, Pydantic

## 🚀 Run Locally

### Backend

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
